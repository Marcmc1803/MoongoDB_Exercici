# MongoDB Exercici - Marc

Aquest projecte és la implementació d'una **Service Layer** amb Mongoose i TypeScript per gestionar una col·lecció de projectes.

## 🚀 Què hi ha?
- **Service Layer**: Tota la lògica CRUD està a `services/`.
- **TypeScript**: Ús d'interfícies per tenir el codi ben lligat.
- **Relacions**: Connecto projectes amb organitzacions fent servir `populate`.
- **Optimització**: Faig servir `.lean()` perquè les llistes vagin més ràpides.

> **Nota**: He fet servir IA únicament per arreglar els errors de tipat de TypeScript, solucionar els problemes amb les extensions dels mòduls de Node.js (ESM) i per ajudar-me a gestionar i presentar millor l'estructura del codi.
